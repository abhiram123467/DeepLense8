## рҹҢҢ What Is This?

Gravitational lensing вҖ” light from distant galaxies bent by massive foreground objects вҖ” is one of astrophysics' most powerful probes of **dark matter** and **cosmology**. But real lensing events are rare and expensive to label.

**DeepLense8 solves the data scarcity problem:** a full DDPM pipeline that generates physically realistic synthetic lensing images, enabling:

- рҹ”¬ Data augmentation for downstream classifiers
- рҹҢ‘ Dark matter substructure simulation at scale
- рҹӣ° Pre-training foundation models for telescope survey data (Rubin LSST, Euclid)

## рҹҸ— Architecture

### DDPM Forward Process

```
Real Image xвӮҖ  в”Җв”Җq(xвӮң|xвӮңвӮӢвӮҒ)в”Җв”Җв–ә  Noisy Image xвӮң
                  (Add Gaussian noise step-by-step over T=1000 steps)

ОІ schedule: Linear  ОІвӮҒ=1Г—10вҒ»вҒҙ вҶ’ ОІвӮң=0.02
ОұМ…вӮң = вҲҸбөўвӮҢвӮҒбө— (1-ОІбөў)     (precomputed cumulative product)

q(xвӮң|xвӮҖ) = N(xвӮң; вҲҡОұМ…вӮң В· xвӮҖ,  (1-ОұМ…вӮң)В·I)   вҶҗ closed-form!
```

### Training Loop

```python
for epoch in range(TRAIN_EPOCHS):
    for x0 in dataloader:                         # real images
        t  = randint(0, T)                         # random timestep
        xt, Оө = ddpm.q_sample(x0, t)              # add noise
        ОөМӮ     = model(xt, t)                       # predict noise
        loss  = MSE(ОөМӮ, Оө)                          # simple L2
        loss.backward()
        clip_grad_norm_(model.parameters(), 1.0)   # stability
        optimizer.step()
    scheduler.step()                               # cosine LR decay
```

### Reverse Process (Sampling)

```python
x = randn(1, 1, 64, 64)           # pure Gaussian noise
for t in reversed(range(1000)):    # 1000 denoising steps
    x = ddpm.p_sample(model, x, t) # gradually remove noise
# x is now a synthetic lensing image рҹ”ӯ
```
